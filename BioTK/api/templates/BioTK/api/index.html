Status legend

release Working, tested, and should not contain major bugs
RC Working, tested or partially tested, possibly has minor bugs, but probably not major bugs
beta Working, but may contain major bugs
alpha Working, but probably contains major bugs
dev Partially working or not working
NYI Not yet implemented (a placeholder)

The API itself (that is, the URLs and parameters) should be considered as subject to change for any status less than RC. At RC and release, the API can be considered as stable and unlikely to change, and all effort will be made to make changes backwards-compatible.

Bug reports are happily accepted for any status above dev, but only release and RC should be considered for serious use, and beta results can be used for less important things, but with care and a critical eye.

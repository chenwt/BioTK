Requirements
============

- xmlstarlet
- libarchive or the multi-threaded equivalent (pkgbuild: mtar-git)
- parallel
- BSD reshape (AUR: openbsd-rs-git) patched to increase BUFSIZ
- cluster3 (pkgbuild: cluster3)
- vowpal wabbit
- lz4
- boost (iostreams)

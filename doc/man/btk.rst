=====
 btk
=====

-------------------------------------
Main command-line interface for BioTK
-------------------------------------

.. include:: common/header.rst

SYNOPSIS
========

  btk 
    <subcommand> <subcommand args>

  btk 
    (for a listing of available subcommands)

DESCRIPTION
===========

This is the main command-line interface to BioTK. You can call any public-facing BioTK script from this script, or you can run this script without arguments to get a list of available subcommands.
